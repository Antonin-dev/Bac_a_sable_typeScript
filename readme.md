## NODE TYPESCRIPT

### Install Typescript
```
npm i typescript --save-dev
```
Compilation et surveillance :
```
"tsc": "tsc",
"watch": "tsc -w"
```
Initialisation du projet :
```
npx tsc --init
```
### *_TsConfig.json_* 
Permet de modifier les parametrage de typescript dans notre projet.
